Описание модуля "Городское освещение"

Данный модуль разработан для мониторинга и управления городским освещением. Работа основана на взаимодействии с ПТК Спрут-М.

Описание методов модуля

ВНИМАНИЕ! МОДУЛЬ УПРАВЛЯЕТ СИЛОВЫМИ ЦЕПЯМИ 0,4kV. БУДЬТЕ ОСТОРОЖНЫ! НЕСАНКЦИОНИРОВАННОЕ ВКЛЮЧЕНИЕ ОБОРУДОВАНИЯ МОЖЕТ ПРИВЕСТИ К ЭЛЕКТРОТРАВМАМ ПЕРСОНАЛА!

  • http://sarov.us.to:9902/citylighting/sprut/show_devices Метод для получения зарегестрированных устройств в ПТК Спрут
  • http://sarov.us.to:9902/citylighting/sprut/state?device_id=16 GET Queryparam. Метод для получения состояния устройства по его id. Возвращает JSON с описанием состояния силового ввода по трём фазам, а также состояния управляющих реле канал1 и канал2
  • http://sarov.us.to:9902/citylighting/sprut/on_evening?device_id=16 GET Queryparam. Метод для активации вечернего режима конкретного устройства.
  • http://sarov.us.to:9902/citylighting/sprut/on_night?device_id=16 GET Queryparam. Метод для активации ночного режима конкретного устройства.
  • http://sarov.us.to:9902/citylighting/sprut/off?device_id=16 GET Queryparam. Метод для перевода в отключенное состояние конкретного устройства.
  • http://sarov.us.to:9902/citylighting/sprut/electric_state?device_id=16 GET Queryparam. Метод для получения электрических параметров устройства по его ID.
